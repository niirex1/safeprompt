# Start here

This is the SafePrompt CCS artifact bundle.

Recommended sequence:
1. Reproduce the manuscript tables from the CSV files:
   - `python scripts/reproduce_tables.py`
2. Verify the outputs against the shipped expected hashes:
   - `python scripts/verify_outputs.py`
3. View the architecture figure:
   - `figures/safeprompt_architecture.png`

No network access is required for steps (1)-(2).
